## 活动简介

这是[极客时间《左耳听风》专栏](https://time.geekbang.org/column/intro/48)中发起的学习活动，要求自发每周完成一个ARTS。

ARTS是以下四项内容的简称：
* **A**lgorithm - 至少做一个leetcode的算法题
* **R**eview - 阅读并点评至少一篇英文技术文章
* **T**ip - 学习至少一个技术技巧
* **S**hare - 分享一篇有观点和思考的技术文章

打卡地址（知乎）：<https://www.zhihu.com/question/301150832>
